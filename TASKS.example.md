# Задачи

> Этот файл читается KrabBoard автоматически.
> Путь по умолчанию: `/root/.openclaw/workspace/TASKS.md`
> Можно переопределить через `tasks_md_path` в config.json.

## Формат таблицы

| Задача | Прогресс | Дедлайн | Приоритет | Статус |
|--------|----------|---------|-----------|--------|
| Разработать лендинг для клиента | 75 | 2026-04-18 | high | in_progress |
| Настроить CI/CD пайплайн | 40 | 2026-04-22 | medium | in_progress |
| Написать документацию к API | 0 | 2026-04-30 | low | todo |
| [x] Провести код-ревью | 100 | 2026-04-14 | medium | done |

---

## Правила

### Колонки

| Колонка | Обязательная | Значения |
|---------|:------------:|---------|
| Задача | ✅ | Любой текст. `[x]` в начале = задача завершена |
| Прогресс | ✅ | 0–100 (можно писать `75` или `75%`) |
| Дедлайн | ❌ | Формат `ГГГГ-ММ-ДД`, иначе игнорируется |
| Приоритет | ❌ | `high` / `medium` / `low` (по умолчанию `medium`) |
| Статус | ❌ | `todo` / `in_progress` / `done` |

### Автоматическое определение статуса

Если колонка **Статус** пропущена или не распознана:
- Прогресс = 0 → `todo`
- Прогресс 1–99 → `in_progress`
- Прогресс = 100 или `[x]` в названии → `done` (скрывается)

### Что скрывается с дашборда

KrabBoard автоматически убирает задачи, у которых:
- Статус = `done`
- Прогресс ≥ 100
- Название начинается с `[x]`
